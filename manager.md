# DeviceWebAPI Managerの設定

DeviceWebAPI Managerを起動します。

設定を選択します。

![](./img/setting001.png)

これから設定の変更の処理をするため、一旦、DeviceWebAPI Managerをオフにします。

![](./img/setting002.png)

![](./img/setting003.png)

赤線で囲まれたチェックボックスを以下のように変更します。

|項目|処理|
|:--|:--|
| 外部IPを許可 | 選択する |
| Local OAuth | 選択をはずす|
| Originの有効化 | 選択をはずす|

![](./img/setting004.png)

![](./img/setting005.png)

設定の変更が完了したら、DeviceWebAPI Managerを再びオンにします。

![](./img/setting007.png)
